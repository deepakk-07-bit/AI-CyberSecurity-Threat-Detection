# 🛡️ AI-Based Cyber Security Threat Detection & Intrusion Prevention System

## 📌 Project Overview

This project is developed as part of the Aarivya Labs Internship Program by H & P Projects.

The system monitors network activities, detects suspicious behavior, logs cyber threats, and provides a real-time dashboard using Machine Learning techniques.

---

## 🚀 Features

- Threat Detection
- Threat Logging
- Machine Learning Based Classification
- Real-Time Dashboard
- FastAPI Integration
- Streamlit Visualization
- Docker Deployment

---

## 🛠 Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit
- FastAPI
- Scapy
- Joblib
